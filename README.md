# PowPingDark

Say yes to darkness on PowPing

![gif](https://media.giphy.com/media/VhK9tEn1uWLUOGvyST/giphy.gif)

# Usage


1. Open [chrome://extensions/](chrome://extensions/)
2. Click "Load unpacked"
3. Add the PowPingDark folder. Done!
